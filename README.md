# MedTrack (Logo Not Yet Designed)

A mini-project designed to help patients manage prescriptions and understand lab reports through timely reminders and AI-powered summaries.


## Problem Statement
Patients often forget to take medicines on time and find lab reports difficult to understand due to complex medical terminology.


## Solution
This system allows doctors to enter prescription schedules, sends automated reminders to patients, and summarizes uploaded lab reports in simple language with general preventive suggestions.


## Features
- Doctor-entered medicine schedules
- Timely medication reminders
- Lab report upload (PDF/Image)
- AI-generated easy-to-understand summaries and Prevention/Curing Recommendations
- Secure authentication & role-based access
- Mobile-friendly interface


## Tech Stack (Changes are expected with further development)
- Frontend: React Native (Expo)
- Backend: Node.js + Express
- Database: PostgreSQL
- Notifications: Firebase Cloud Messaging
- AI: OpenAI API
- Security: JWT, bcrypt, HTTPS
